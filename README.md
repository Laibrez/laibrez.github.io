# Laisha Bravo Portfolio

A modern, responsive portfolio website for Laisha Bravo.

## Features

- **Responsive Design**: Fully responsive layout that works on all devices
- **Modern UI**: Clean and professional design with smooth animations
- **Sections**:
  - Hero section with call-to-action buttons
  - About section with skills showcase
  - Portfolio section with project grid
  - Contact section with contact form and information
- **Interactive Navigation**: Smooth scrolling and mobile-friendly menu
- **Customizable**: Easy to update content and add your own images

## Getting Started

1. Open `index.html` in a web browser to view the portfolio
2. The website is ready to use with placeholder content

## Customization

### Adding Your Images

Replace the placeholder image references in the `images/` folder:

- `profile.jpg` - Your profile photo for the About section
- `project1.jpg` through `project6.jpg` - Images of your work/projects

### Updating Content

Edit `index.html` to customize:

- **Personal Information**: Update your name, description, and contact details
- **About Section**: Modify your bio and skills
- **Portfolio Items**: Add or remove projects, update titles and descriptions
- **Contact Information**: Update email, phone, and social media links

### Styling

Edit `styles.css` to change:

- **Colors**: Modify CSS variables in `:root` at the top of the file
- **Fonts**: Change the Google Fonts import in `index.html`
- **Layout**: Adjust spacing, sizes, and grid layouts
